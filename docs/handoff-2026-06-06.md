## 🖼️ 이미지/시각화 개선 필요 (NEXT)

### 1. Landscape 2D contour — 계층 구분 모호 (우선)
- **증상**: 2D가 블러처럼 뭉개져(덩어리) PCC가 한 등고선 안에 있는 것처럼 보여 **커뮤니티별 구분 안 됨** (3D는 구분되는데 2D는 안 됨).
- **원인 추정**: `contours.coloring:"heatmap"`(연속 채움) + `line.width:0`(등고선 선 없음) + kernel σ=0.05 평활 → 부드러운 히트맵.
- **개선 방향(actionable, Landscape.tsx build2D)**:
  - `coloring:"fill"`(이산 밴드) + **등고선 선 그리기**(`line.width≥0.5`, 옅은 회색).
  - 레벨 세분화: 현재 `ncontours:100`(너무 촘촘해 연속처럼 보임) → **이산 레벨 ~10–15개**로(`contours.size`를 (2*zlim)/12 정도) 계층이 또렷하게.
  - 필요시 kernel σ를 0.04로 줄여 덩어리 완화 / zlim 대신 실제 데이터 분위수 기반 레벨 고려.
  - 목표: 3D처럼 커뮤니티별 봉우리/골이 2D 등고선에서도 층층이 구분.
